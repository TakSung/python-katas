# 🎯 HiddenNumber

숫자가 숨겨져 있다!  
당신의 임무는 **컴퓨터가 마음속에 숨긴 숫자를 찾아내는 것**입니다.  
직관력과 논리를 이용해 정답에 점점 다가가 보세요.

---

## 🧩 프로젝트 개요

**HiddenNumber**는 Python의 `Tkinter` GUI 라이브러리를 활용한 숫자 추리 게임입니다.  
1부터 100 사이의 무작위 숫자가 정답으로 설정되며,  
사용자는 입력창을 통해 숫자를 추측하고 결과 메시지를 확인합니다.

---

## 🧠 학습 목표

이 코딩 카타를 통해 다음 내용을 실습합니다:

| 영역 | 학습 내용 |
|------|------------|
| **Python 문법** | 조건문(`if`), 반복문(`while`), 변수 사용 |
| **GUI 프로그래밍** | `Tkinter`의 `Label`, `Entry`, `Button` 활용 |
| **이벤트 처리** | 버튼 클릭 시 함수 실행 (`command` 속성) |
| **상태 관리** | 시도 횟수 기록 및 메시지 업데이트 |

---

## 🎮 게임 규칙

1. 프로그램을 실행하면 **1부터 100 사이의 숫자**가 무작위로 선택됩니다.  
2. 사용자는 입력창에 숫자를 입력하고 **[Guess]** 버튼을 누릅니다.  
3. 컴퓨터는 다음 중 하나의 메시지를 GUI로 보여줍니다:
   - “더 큰 수입니다!”
   - “더 작은 수입니다!”
   - “정답입니다!”
4. 정답을 맞히면 시도 횟수와 함께 축하 메시지가 표시됩니다.

## 🧑‍💻 개발자 팁

* `entry.get()`으로 입력값을 문자열로 받으니 `int()` 변환을 잊지 말자.
* GUI 내에서 Label을 바꿀 때는 `label.config(text="새 문장")`을 사용한다.
* 프로그램 실행 시 마다 `random.randint(1, 100)`으로 새로운 정답이 설정된다.
